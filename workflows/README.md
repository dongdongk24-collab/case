# Workflows

각 업무의 단계별 지시서 (업무 매뉴얼).

## 형식

각 workflow 파일은 아래 항목을 포함해:

- **목표**: 이 workflow가 뭘 하는지
- **입력값**: 실행 전에 필요한 것
- **단계**: 순서대로 실행할 Tool과 행동
- **결과물**: 완료 후 어떤 상태여야 하는지
- **문제 대응**: 에러가 나면 어떻게 하는지

## 규칙

- 새 workflow는 사장 허락 없이 만들거나 삭제 금지
- 더 나은 방법 발견 시 업데이트 OK
- 반복되는 업무는 반드시 workflow로 만들기
